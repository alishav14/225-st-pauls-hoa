How this site works

This website is a static site hosted on GitHub Pages. Updates are as easy as editing text files in GitHub or VS Code and pushing changes.

  • Fast loading and mobile-friendly
  • Low cost (domain only)
  • No database or server maintenance

Editing announcements

Announcements are stored in assets/js/main.js. Add a new announcement in one place and it shows on the homepage.
